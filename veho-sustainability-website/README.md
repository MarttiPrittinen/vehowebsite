# Veho Sustainability Prototype Website

This is a static prototype website for a school/computer engineering assignment.

## How to open in VS Code

1. Download and unzip the folder.
2. Open the folder in Visual Studio Code.
3. Open `index.html` in a browser.

Recommended: install the VS Code extension **Live Server**, then right-click `index.html` and choose **Open with Live Server**.

## How to publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository Settings > Pages.
4. Select branch `main` and root folder.
5. Copy the published GitHub Pages link and submit it.
